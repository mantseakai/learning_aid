💡 What is Programming?

Programming is giving instructions to a computer to make it do something!

In Scratch, we use blocks to give instructions to a character called a sprite 🐱

🧱 What is a Block?

A block is a small instruction used in a program.
Blocks are joined together to make a program!

👆 Tap the blocks below to see what they do!

🖱️ How do we use Scratch?
1
Use the mouse to drag and move blocks
2
Join blocks together like puzzle pieces
3
Click the green flag 🚩 to run your program
4
Watch your sprite follow every instruction!
